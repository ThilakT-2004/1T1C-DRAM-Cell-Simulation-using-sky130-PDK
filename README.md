# 1T1C-DRAM-Cell-Simulation-using-sky130-PDK
SPICE simulation of a 1-Transistor 1-Capacitor (1T1C) DRAM cell using the open-source sky130 180nm PDK in ngspice. Includes DC stability analysis (butterfly curve / sense amplifier trip point) and transient analysis for write and read operations. Designed and simulated using KiCad schematic + ngspice.
